# Как удалить тестовые объявления в Supabase

Тестовые объявления созданы скриптом из `supabase_full_new_project.sql` и привязаны к тестовым пользователям.

## В каких таблицах лежат данные

- **`listings`** — сами объявления (заголовок, цена, год, марка, модель и т.д.).
- **`listing_images`** — ссылки на фото объявлений (у каждого объявления может быть несколько записей). Связана с `listings` по полю `listing_id`; при удалении объявления записи в `listing_images` удаляются автоматически (CASCADE).

## Как удалить в интерфейсе Supabase

1. Откройте **Supabase Dashboard** → ваш проект → **Table Editor**.
2. Откройте таблицу **`listings`**.
3. Найти тестовые объявления можно по:
   - **user_id**: `a0000001-0000-4000-8000-000000000001`, `a0000002-0000-4000-8000-000000000002`, `a0000003-0000-4000-8000-000000000003`;
   - или по **title**: например, «Toyota Camry 2022», «BMW 3 Series 2020», «Mercedes-Benz C-Class 2021».
4. Отметьте нужные строки и нажмите **Delete** (или откройте строку и удалите). Записи в **`listing_images`** для этих объявлений удалятся автоматически.

## Как удалить через SQL Editor

В **SQL Editor** выполните:

```sql
-- Удалить тестовые объявления (фото удалятся автоматически из listing_images)
DELETE FROM listings
WHERE user_id IN (
  'a0000001-0000-4000-8000-000000000001',
  'a0000002-0000-4000-8000-000000000002',
  'a0000003-0000-4000-8000-000000000003'
);
```

Так удалятся все объявления, созданные от имени этих трёх тестовых пользователей. Записи в **`listing_images`** для этих объявлений удалятся каскадно.

Если нужно удалить только фото по конкретным объявлениям (не удаляя сами объявления), можно удалять строки в **`listing_images`** по `listing_id` нужных объявлений.
